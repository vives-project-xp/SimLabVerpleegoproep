# WebsiteLayout

Deze map is alleen bedoeld voor de website en PWA van het SimLab verpleegoproepsysteem.

Gebruik de PWA-map voor de webapp:

```text
webbsiteLayout/
`-- PWA/
    |-- README.md
    |-- backend/
    `-- frontend/
```

De uitleg in deze map gaat enkel over:

- frontend;
- backend;
- API;
- login;
- push notifications;
- PWA-bestanden;
- website troubleshooting;
- deployment van de website.

Uitleg over hardware, Home Assistant, Zigbee, MQTT en ledstrips hoort in de andere documentatiemappen of repositories.
